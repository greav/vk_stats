This application is intended to receive data and statistics about posts on the wall of the VK user or VK community by ID and date from which the data should be taken into account.

On the download page, you can enter user ID (or user domain) or integer community ID with minus (or community domain) and date, and then select the required fields that need to be downloaded. The file will be received in the CSV format.

The following links to attachments are extracted: photos, videos, documents, articles and reposts.

On the statistics page for the entered user ID (or user domain) or integer community ID with minus (or community domain) and date, you can get statistics on the number of posts and the average number of likes/reposts/comments by:

  • hours
  • days of the week
  • months
  • years

Results will be obtained in accordance with Moscow time.
